# Brachiopod Diversity Dynamics

This repository contains the complete R code pipeline used to analyze the macroevolutionary diversity, ecological niche displacement, and paleolatitudinal reorganization of Brachiopoda across the Mesozoic and Cenozoic.

To overcome historical biases, this pipeline integrates global fossil occurrence data from the Paleobiology Database (PBDB) and regional data from the Geobiodiversity Database (GBDB). It applies stringent coverage-based rarefaction and extrapolation methodologies alongside a strict ecological filtering protocol to conduct symmetrical comparisons between brachiopods and an ecologically equivalent guild of bivalves and echinoderms.

## Pipeline Overview

The analytical pipeline is modular and divided into eight sequential scripts. They must be executed in numerical order, as subsequent scripts depend on the data outputs and workspace archives generated by previous ones:

1. **`01_Data_Fetching_and_Merging.R`**: Downloads raw Mesozoic-Cenozoic occurrence data for Brachiopoda, Bivalvia, and Echinodermata from the PBDB via API, and integrates locally stored GBDB fossil records.
2. **`02_Data_Cleaning_and_Unification.R`**: Cleans taxonomic nomenclature, resolves synonyms using fuzzy matching, unifies PBDB & GBDB spatial/temporal data, assigns strict international stratigraphic stages, removes cross-database duplicates, and filters baseline environments.
3. **`03_Ecological_and_Comparative_Processing.R`**: Harmonizes ecological vocabulary between the two databases. Implements Genus-level Trait Imputation (taxonomic inheritance) to rescue occurrences with missing ecological traits (NA). Exports the cleaned Brachiopoda dataset and extracts an ecologically equivalent guild (epifaunal, stationary suspension feeders) for Bivalvia and Echinodermata.
4. **`04_Brachiopoda_Diversity_Analysis.R`**: Calculates and visualizes coverage-based standardized diversity (Hill number $q=0$, interpolated/extrapolated) for Brachiopoda at both stage-level and 10-Myr bins. Analyzes both the ecologically filtered dataset and the global unfiltered baseline.
5. **`05_Clade_Specific_Evolution.R`**: Mirrors the analytical logic of Script 04 to separately calculate and plot standardized diversity trajectories for the comparative clades (Bivalvia and Echinodermata).
6. **`06_Consolidate_and_Archive_Diversity.R`**: Consolidates the standardized statistical results across all clades, performs strict sample coverage quality filtering (excluding bins where SC < 0.6), and archives the workspace for downstream macroecological analyses
6. **`07_Integrated_Macroevolution_Niche.R`**: Generates the final visualization pipeline for the manuscript, focusing on Brachiopoda. Produces multi-panel plots for taxonomic scale dependence (species vs. genus vs. family), ecological niche occupation (habitat proportions), and paleolatitudinal distribution shifts (density ridge plots).
7. **`08_Combined_Main_Plots.R`**: Combines the Brachiopoda and Bivalvia diversity curves into side-by-side comparative panels for the manuscript's main figures, ensuring identical plot logic and aesthetics (Echinodermata excluded from main text due to sparse data).

## Prerequisites
To run this code, you will need R installed along with the following packages:
* `tidyverse`
* `iNEXT`
* `divDyn`
* `stringdist`
* `ggplot2`

## Usage
1) Clone this repository to your local machine.
2) Set your working directory in R to the project folder (e.g., setwd("D:/PBDB_Project")).
3) Ensure your local raw GBDB data files are placed in the root directory.
4) Run the scripts sequentially (01 through 08).
   Intermediate datasets (e.g., .rds files) and analytical results will be automatically saved to designated subfolders (e.g., /Plot_Data, /Main_Figures, /Supplementary_Figures) generated by the scripts.
